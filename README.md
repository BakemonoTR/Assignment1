## File Organization

* **`index.cshtml`**: The home page of the website. It currently features a curated list of **"This Week's Popular Books"**.
* **`about.cshtml`**: A descriptive page that introduces the website, its mission, and the team behind it.
* **`services.cshtml`**: This page lists the various services offered by the blog along with their **prices**
* **`contact.cshtml`**: Includes a **contact form** for users to send messages or inquiries directly to the administration.
* **`assets/`**: A centralized folder containing all media files, including book covers, icons.
